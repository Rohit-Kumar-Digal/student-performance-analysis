# 📊 Student Performance Analysis

## 📌 Project Overview

This project analyzes student performance data using Python (Pandas & NumPy).
The goal is to extract insights such as average marks, department-wise performance, and score distribution.

---

## 🛠️ Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Google Colab

---

## 📂 Dataset

The dataset contains:

* Student Name
* Department
* Marks in different subjects
* Total Score

---

## 📊 Key Analysis

* Average marks by department
* Distribution of total marks
* Performance comparison
* Data cleaning and preprocessing

---

## 📈 Sample Insights

* Most departments have similar average scores
* Score distribution shows majority between 240–280
* Few students scored below average

---

## 🚀 How to Run

1. Open the notebook in Google Colab
2. Run all cells step by step
